I. MY MUSIC - SIMPLE WEB MUSIC PLAYER
  - A Lightweight, Browser‑Based Music Player Built With HTML, CSS, And JavaScript.
  - Designed To Deliver A Clean Interface, Fast Performance, And An Easy Listening Experience.

II. FEATURES
  - Unlimited Tracks Upload.
  - Track Searching For Quick Navigation.
  - Play One Track Mode.
  - Replay Track Mode.
  - Shuffle Track Mode.
  - Theme Mode Switch (Dark Mode And Light Mode).
